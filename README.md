# 成绩查询系统

## 项目简介

成绩查询系统是一款面向高中学段学校/教育机构/个人开发者使用的轻量级成绩查询管理系统。系统提供前台考生成绩查询和后台数据管理两大核心功能，帮助学校高效管理考生成绩数据，并为考生提供便捷的成绩查询服务。

## 核心功能

### 前台查询
- 考生通过姓名和准考证号查询个人高考成绩
- 展示各科成绩明细、总分、全市位次及批次线
- 支持站点标题、副标题、Logo 等个性化设置
- 响应式设计，支持手机、平板、电脑多端访问

### 后台管理
- 考生信息管理（新增、编辑、删除）
- 各科成绩录入与管理
- 排名信息维护
- 系统设置（站点标题、考试信息、页脚信息、Logo 上传）
- 数据统计概览（考生总数、今日新增、平均总分）
- 登录频率限制与 CSRF 防护

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | PHP 7.4+ |
| 数据库 | MySQL 5.7+ / MariaDB |
| 前端 | 原生 HTML5 + CSS3 + JavaScript |
| 架构 | 前后端分离，RESTful API |

## 安装部署

### 环境要求
- PHP 7.4 或更高版本
- MySQL 5.7 或更高版本 / MariaDB 10.2+
- Apache / Nginx 服务器
- 支持 URL Rewrite（可选）

### 安装步骤

1. 将项目文件上传至服务器
2. 访问 `域名/install` 进入一键安装向导
3. 填写数据库连接信息和管理员账号
4. 安装完成后**立即删除 `install/` 目录**
5. 访问前台查询页面和后台管理页面

## 使用说明

### 前台查询
1. 考生访问系统首页
2. 输入姓名和准考证号
3. 点击查询，查看成绩结果

### 后台管理
1. 访问 `域名/admin` 进入后台
2. 使用管理员账号登录(install安装时填写的）
3. 在考生管理中添加或导入考生数据
4. 在系统设置中配置站点信息

## 安全特性

- CSRF 令牌防护
- 登录频率限制（5 次失败锁定 15 分钟）
- 会话安全（HttpOnly、Secure、SameSite）
- SQL 注入防护（PDO 预处理）
- XSS 防护（输出转义）
- 文件上传类型校验
- 安全响应头（X-Frame-Options、CSP 等）
- IP 查询限流
